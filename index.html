  3. Prepare the cheese mixture:
    1. In a separate bowl, mix together ricotta cheese and parsley.
  4. Layer & bake

Dart Code

Here is a snippet of Dart code that creates a data class(based on 75 as the two last digits of my S number):

        
            @immutable
            class User {
            final String name;
            final int age;
            
            User(this.name, this.age);
            
            User copyWith({
                String name,
                int age,
            }) {
                return User(
                name ?? this.name,
                age ?? this.age,
                );
            }
            
            @override
            bool operator ==(Object o) {
                if (identical(this, o)) return true;
            
                return o is User && o.name == name && o.age == age;
            }
            
            @override
            int get hashCode => name.hashCode ^ age.hashCode;
            }
